📥 Importar Produtos via Planilha

Exporte seus produtos do Clique Fácil em formato CSV ou Excel e importe aqui. O sistema irá adicionar produtos novos e atualizar preços e estoque dos existentes automaticamente.

💡 O sistema identifica produtos pelo campo Referência. Se já existir um produto com a mesma referência, o preço e o estoque serão atualizados. Caso contrário, será criado um novo produto.

Prévia dos dados

📋 Como exportar do Clique Fácil

1

Acesse o Clique Fácil

Entre no seu painel do Clique Fácil e vá até o menu Produtos ou Estoque.

2

Exporte a listagem

Procure o botão Exportar ou Baixar planilha e escolha o formato CSV ou Excel.

3

Ajuste as colunas (se necessário)

O sistema aceita qualquer coluna chamada nome, marca, preco, estoque, referencia, categoria. Você pode renomear as colunas da planilha para esses nomes.

4

Importe aqui

Arraste o arquivo para a área acima. O sistema mostrará uma prévia antes de confirmar. Após confirmar, o catálogo é atualizado automaticamente.

⚙️ Configuração do Supabase

Conecte o painel ao seu banco de dados Supabase. Após configurar, todos os produtos serão salvos e sincronizados automaticamente.

Encontre em: Supabase Dashboard → Settings → API → Project URL
Encontre em: Supabase Dashboard → Settings → API → anon public

Produtos em destaque da Home

Edite somente os cards da pagina inicial: marca, titulo, foto/video e link de destino.

💡 O link pode ser um produto especifico, por exemplo produto.html?id=10, ou qualquer pagina do site.
